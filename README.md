🌍 About the Project
✨ Sahayta is a real-time community help platform that bridges the gap between people in need and nearby volunteers using live location tracking.
💡 Built with the vision:

“हर ज़रूरत मायने रखती है। हर स्वयंसेवक ज़रूरी है।”

🚨 Problem Statement
❌ People struggle to find immediate help in emergencies
❌ Volunteers don’t know where help is needed
❌ No real-time coordination system
👉 Result: Missed opportunities to help when it matters most


💡 Solution
✅ Sahayta provides a live map-based system where:
Needy users can instantly request help
Volunteers can locate and assist nearby people
Safe verification ensures trust

✨ Key Features
👤 For Users (Need Help)
📍 Live GPS location sharing
🆘 One-tap SOS alert
📝 Easy request submission
📊 Track request status
🔐 QR-based secure verification

🤝 For Volunteers
🗺️ Real-time map of nearby requests
📌 Accept & respond instantly
💬 Built-in chat system
🧭 Navigation support
🏆 Leaderboard & rewards
